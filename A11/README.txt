STEP-BY-STEP:

1. Create your folder with individual/team identification.
* Suggestion: First name + Last 3 Id numbers
* Ex: > mkdir Paulo081

2. Create the Git folder:
* Command: > git init

3. Create the A11 folder:
* Command: > mkdir A11

4. Add this folder to your local repository:
* Command: > git add A11

4. In this folder, download the "CST8152_Compilers_A11_AnswerTemplate.docx" from A11 assignment activity.

5. Rename this file to a language name.
* Requirement: Give a snake name (in the case of GPL) or a context name (in the case of DSL).
* Ex: I will update "Boa" to "Jiboia" (brazilian name to "Boa"): CST8152_Compilers_A11_Jiboia.docx

6. Then, I can add this file and folder in git:
* Command: > git add .

7. Include this file in your stage:
* Command: > git commit -m "Initial A11"

8. Now, you can start changing this file. Periodically update it in the repository, repeating the commit command.
